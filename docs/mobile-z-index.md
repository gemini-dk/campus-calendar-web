# mobile画面のz-index順序

モバイル画面では複数のダイアログが表示されるが、前後関係をZ-indexでコントロールしている。

## z-20
- FABなど

## z-30
- DailyCalendar画面(週・月カレンダーから日付選択時の画面)

## z-35
- 授業活動ダイアログ

## z-40
- メイン画面下部のタブ

## z-45
- 右側ハンバーガーメニュー、そこから開く設定画面

## z-50
- 課題・メモ作成ダイアログ
- 授業作成ダイアログ
- 授業スケジュール作成ダイアログ
- スケジュール調整ダイアログ
- カレンダー設定確認ダイアログ
